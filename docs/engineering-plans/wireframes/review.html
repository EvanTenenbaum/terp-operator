10 Design Rules (v1.1) — Click to expand

Derived from live Mercury demo app tokens (§14) + TERP feature mapping (mapping doc) + domain QA against TERP's 27-view, 8-status-per-entity complexity. Full source: §17.9. Separate file: DESIGN-RULES.md. Applies to all Phase 0+ implementation.

#RuleSourceApplies To
1 Content-first, chrome-second — Data dominates. Chrome recedes via shadow-only cards and 0.08-0.12 opacity borders. Mercury IA: 3 elements in main area, table at 75%+ visual weight All GridView, MasterDetailView, DashboardView
2 Progressive disclosure — Bulk actions, detail panels, filter popovers, collapsible sections appear on demand. Mercury: no bulk bar or detail panel visible by default BulkActionBar, DetailSlideover, FilterToolbar
3 Subtle tab pills — Status navigation as Mercury-style pill toggles with count badges. NOT heavy framed tabs. TERP domain need (8 statuses) + Mercury pill style ViewTabBar, all view-level status tabs
4 Glanceable metrics — 3-5 subtle metric cards above tables. Expandable on click. Shadow-only cards. Adjusted from Mercury's single KPI line GridSummaryStrip, Dashboard KPI section
5 Minimal view headers — View title + subtle action buttons. No heavy framing. 27 views need orientation. TERP domain need + Mercury minimalism All view templates
6 Table dominance — Table occupies 70-80% of visual space. Supporting context only. Mercury: 968px table in 1440px viewport GridView, MasterDetailView
7 Invisible chrome — Borders at 0.08-0.12 opacity. Shadows for depth. Cards: shadow-only. Mercury cards: no border, shadow-only. Pill borders at 0.16 opacity. All components, all views
8 Typographic hierarchy — Inter/system-ui, 400-600 weight. No uppercase headers. Table headers: 13px regular case. Mercury: max weight 480, no bold. Adjusted for TERP data density. All typography
9 Collapsible supporting sections — Multi-section views show one primary surface. Others are collapsible. TERP: SalesView's 6 panels → 1 primary + collapsible + slide-over SalesView, IntakeView
10 Clear error states — #d92d20, inline, contextual. Cell: left red border. Bulk: partial result bar. Empty: centered CTA. Adjusted from Mercury berry-red for accessibility ComboboxCellEditor, BulkActionBar, empty states
