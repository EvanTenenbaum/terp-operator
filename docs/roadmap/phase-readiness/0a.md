# Phase 0a Readiness — Foundation Without Public IA Churn

Date: 2026-05-12
Status: ready for implementation planning

## Purpose

Create the lowest-risk foundation for the design system without changing where operators go today. Phase 0a is a cleanup and focus pass, not a new product surface.

## Required Reading

- `docs/product/north-stars.md`
- `docs/product/work-loops.md`

- `docs/design/spec.md`
- `docs/design/replication-playbook.md`
- `docs/roadmap/2026-frontend-direction-roadmap.md`

## Prerequisites

- Current app runs from existing routes.
- `pnpm audit:parity` is green before work starts.
- No backend schema or command-catalog changes are bundled into this phase.

## Scope

| Item | Product reason | Capability IDs | Recipe |
| --- | --- | --- | --- |
| WorkspacePanel focus revision | Reduce always-visible clutter while preserving grid-first work. | CAP-006 | R4, R11 |
| CommandPalette cleanup | Make search/run feel like support infrastructure, not daily workflow replacement. | CAP-015 | R9, R12 |
| Hotkeys groundwork | Preserve keyboard-first behavior before canvas grammar changes. | CAP-006 | R5, R14 |

## Out Of Scope

- New route.
- New backend command.
- New query.
- New top-level navigation grouping.
- New component library, font, CSS-in-JS, or router.
- Routine modal wizard.

## Feature Flag

No public feature flag required. Changes are internal component behavior and should be reversible by git revert.

## Pre-Flight Checks

```bash
pnpm typecheck
pnpm audit:parity
pnpm audit:product-roadmap
```

## Acceptance Evidence

- Operators can still access every existing route.
- Existing command surfaces remain reachable through palette, row actions, or current buttons.
- Default visible button count does not increase on any touched surface.
- Keyboard shortcuts do not fire while editing a text input.
- No visible raw JSON appears in ordinary operator views.

## Smoke Test

Use the Replication Playbook 10-item smoke test, plus:

1. Open Dashboard, Intake, Sales, Payments, Inventory.
2. Confirm row selection still exposes the correct primary action.
3. Press `Cmd+K`, search a command, close with `Esc`.
4. Edit a grid cell and press Enter; hotkeys must not steal focus.

## Rollback

Revert Phase 0a component changes. No data migration or backend rollback should exist.

Small to medium. One PR preferred.

Frontend integration engineer with product PM review.
